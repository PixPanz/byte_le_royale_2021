==========
Visualizer
==========

.. image:: ./visualizer.png
  :width: 400

The Visualizer is an easier way to view and debug your code. It reads information from the turn logs and translates that into visuals. You can now view your complete destruction in beautiful graphics made by our very own Sander Eggum. 


##################
Controls
##################

You can pause the game using 'p' and select items from the paused screen with the spacebar. You can also speed up the game with the left and right arrow keys.
